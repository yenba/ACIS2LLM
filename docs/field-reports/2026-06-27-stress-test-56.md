- **Category:** data_quality
- **Severity:** medium
- **Description:** The agent reports 'all 15 Julys have complete 31-day records' for the period 2011-2025. If the current date is before July 31, 2025, then July 2025 data would be incomplete. The agent may have included a partial month in its count without flagging this to the user. For frequency analysis questions, including incomplete months can skew results.
- **Suggestion:** Add a gotcha to SKILL.md: 'When analyzing multi-year frequency patterns, always check if the most recent month in the requested period is complete. If the current date is before the end of that month, either exclude it from the count or add a disclaimer. The get_single_station_acis_data function may return partial data without warning.'
- **Pointer:** `SKILL.md - gotchas section`
