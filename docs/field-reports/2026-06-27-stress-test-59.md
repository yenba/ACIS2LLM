### Issues

#### 2025 data may be incomplete/partial

- **Category:** data_quality
- **Severity:** low
- **Description:** The agent included 2025 in the 10-year range (2016-2025). If this query was run before May 2025 ended, the 2025 GDD50 value would be based on partial month data, which could mislead the user about typical conditions. The agent didn't flag this as potentially incomplete data.
- **Suggestion:** Add a note in SKILL.md under 'Gotchas' or 'Best Practices' about handling current-year data: 'When requesting data for the current year, be aware that monthly totals may be incomplete if the month has not yet ended. Consider flagging current-year data as preliminary or excluding it from historical averages.'
- **Pointer:** `SKILL.md`

#### GDD calculation method not documented in response

- **Category:** doc_gap
- **Severity:** low
- **Description:** The agent returned GDD50 values but didn't specify the calculation method used (e.g., standard single sine, simplified average, etc.). Different GDD calculation methods can yield slightly different results, which could matter for agricultural planning.
